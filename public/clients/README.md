# Client project pages

These static client overviews (e.g. Manan Grocery, Ratanshi Kheraj Sarees) are served by the **dashboard** app.

**Canonical base URL (use this for all client links):**

**https://dashboard.aiora.live/clients/**

Examples:
- Manan: https://dashboard.aiora.live/clients/manan-grocery/
- Bhavesh (Ratanshi Kheraj): https://dashboard.aiora.live/clients/bhavesh-sarees/

Do **not** use `aiora.live` or `www.aiora.live` for client pages — those domains serve the marketing site. Client HTML lives in this repo’s `public/clients/` and is only available at **dashboard.aiora.live**.
