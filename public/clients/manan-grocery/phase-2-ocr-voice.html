📦 What We Need From Manan

For OCR and Voice to work, we need to match extracted items against Manan's actual product catalog.

  • Complete Product List (with prices) Required From Manan
    Every product Manan sells with: name, price, available sizes/weights, category. Can be an Excel sheet, Google Sheet, PDF, or even photos of the rate card — we'll digitize it.
  • Common Local Names / Aliases Required From Manan
    Local/Hindi names customers commonly use. E.g., "Tel" = Cooking Oil, "Besan" = Gram Flour, "Maida" = All-Purpose Flour, "Doodh" = Milk. We'll build an alias mapping for fuzzy matching.
  • 5–10 Sample Grocery Lists Required From Manan
    Real grocery lists from actual customers (photos of handwritten notes, WhatsApp messages, etc.). These help us train and test the OCR accuracy for Manan's specific customer base and handwriting styles.
  • Typical Customer Languages From Manan
    Do customers usually speak in Hindi, English, or Hinglish? Do any write lists in regional languages? This helps us tune the OCR and voice models.
  • Default Quantities / Sizes From Manan
    When someone says "aata" without specifying a size, what should be the default? E.g., Atta defaults to 5kg, Oil to 1L, Dal to 1kg. These help resolve ambiguous orders.
  • Brand Preferences From Manan
    Does Manan stock specific brands? If customer says "oil", does he send Fortune, Saffola, or loose? We need default brand rules for common items.

🔧 What AIORA Builds

All technical work is handled by us.

OCR Pipeline

Image download from WhatsApp → GPT-4o Vision for text extraction (handles handwritten Hindi/English) → Structured item+quantity parsing → Fuzzy catalog matching with alias support → Order summary generation.

Voice Pipeline

Audio download from WhatsApp → OpenAI Whisper transcription (Hindi/English/Hinglish) → GPT-4o for structured item extraction → Same fuzzy matching and order flow as OCR.

Smart Matching Engine

Fuzzy search with alias mapping. Handles misspellings ("aata" / "atta" / "आटा"), brand inference, quantity normalization ("ek kilo" → 1kg), and ambiguity resolution. Unmatched items are flagged for the customer to clarify.

Order Confirmation Flow

After matching, sends a formatted order summary on WhatsApp. Customer can confirm, edit items, or cancel. Confirmed orders go into Manan's order queue.

About Accuracy

OCR on handwritten lists is ~85-95% accurate depending on handwriting clarity. Voice notes are ~95%+ accurate for Hindi and English. The system always asks for confirmation before placing an order, so any misreads are caught before checkout. Accuracy improves over time as we tune the alias mappings.
